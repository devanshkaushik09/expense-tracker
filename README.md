# 💰 Expense Tracker (MERN)

A secure full-stack **MERN application** for managing income and expenses with a clean Material-UI interface.  
Track your finances, visualize insights with interactive charts, and export data for offline use.

---

## 🚀 Features

- 🔐 **JWT Authentication** – Secure login & signup  
- 📊 **Interactive Charts** – Visualize income vs. expenses  
- 💾 **MongoDB Backend** – Persistent storage for financial records  
- 📤 **Excel Export** – Download data for offline use  
- 🎨 **Material-UI** – Modern responsive design  
- ⚡ **Full-Stack MERN** – React, Node.js, Express, MongoDB

## 📦 Installation

### Clone the repo
```bash
git clone https://github.com/devanshkaushik09/expense-tracker.git
cd expense-tracker
**Backend Setup**
cd backend
npm install
npm run dev

**Frontend Setup**
cd frontend
npm install
npm run dev

**ENV**
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
